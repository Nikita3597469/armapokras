# Подключение домена (Рег.ру) к Vercel

Сайт АрмаПокрас задеплоен на Vercel. Чтобы ваш домен открывал этот сайт:

## 1. Добавить домен в Vercel

1. Зайдите на [vercel.com](https://vercel.com) → ваш проект **armapokras**.
2. Откройте **Settings** → **Domains**.
3. Нажмите **Add** и введите ваш домен (например `armapokras.ru` или `ваш-домен.ru`).
4. Добавьте оба варианта: **ваш-домен.ru** и **www.ваш-домен.ru**.
5. Vercel покажет, какие записи нужно создать (ниже — типовой вариант).

## 2. Записи DNS в Рег.ру

В кабинете Рег.ру на странице управления доменом (как на вашем скриншоте):

### Удалите или измените старые A-записи

- Сейчас у вас: `A @ → 95.163.244.138` и `A www → 95.163.244.138`.  
  Их нужно **удалить** (иконка корзины) или **изменить** значения на те, что ниже.

### Добавьте новые записи («+ Добавить запись»)

Используйте **точные значения из вашего Vercel** (Settings → Domains):

| Тип   | Имя (хост) | Значение |
|-------|------------|----------|
| **A** | `@`        | `216.198.79.1` |
| **CNAME** | `www`  | `f728b0375d34ac37.vercel-dns-017.com.` |

- **Для корня (@):** тип **A**, имя **@**, значение **216.198.79.1**.
- **Для www:** тип **CNAME**, имя **www**, значение **f728b0375d34ac37.vercel-dns-017.com.** (с точкой в конце, как в Vercel).

TTL можно оставить по умолчанию (например 3600) или 60.

## 3. Проверка

- В Vercel во вкладке Domains статус домена должен стать **Valid** (иногда через 5–15 минут, иногда до 24–48 часов).
- Откройте в браузере **ваш-домен.ru** и **www.ваш-домен.ru** — должен открываться сайт АрмаПокрас.

## 4. Если Рег.ру не даёт CNAME для www

Некоторые панели требуют в качестве значения только домен без `cname.`. Тогда для **www** укажите: **cname.vercel-dns.com.** (с точкой в конце).

---

**Итого в Рег.ру:**  
1) A‑запись: **@** → **216.198.79.1**  
2) CNAME‑запись: **www** → **f728b0375d34ac37.vercel-dns-017.com.** (с точкой в конце)

После сохранения подождите распространения DNS (обычно до 24 часов).
